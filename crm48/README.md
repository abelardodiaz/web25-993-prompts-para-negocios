## Metodologia

Este proyecto sigue la metodologia de los **Prompts Maestros** para evaluar ideas de negocio:

1. **Validador (Prompt 1):** Analizar mercado, competencia, validar problema
2. **Arquitecto (Prompt 2):** Modelo financiero, pricing, unit economics
3. **CMO (Prompt 3):** Plan de traccion, primeros 100 clientes
4. **Constructor MVP (Prompt 3.5):** Plan de ejecucion de bajo costo
5. **Inversionista (Prompt 4):** Analisis tipo VC, investment memo
6. **Compilador (Prompt 5):** Pitch deck final
