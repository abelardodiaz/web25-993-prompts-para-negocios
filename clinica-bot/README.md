## Checklist P1 - Validador de Ideas

### Desk Research (Fase A)
- [x] Tamano de mercado (TAM/SAM/SOM)
- [x] Analisis competitivo (6 competidores mapeados)
- [x] Perfil cliente ideal ("Dr. Carlos")
- [x] Costos WhatsApp API calculados
- [x] Hipotesis definidas (H1-H5)

### Entrevistas de Descubrimiento (Fase B)
- [ ] Lista de 30 consultorios con contacto
- [ ] 10+ entrevistas completadas
- [ ] 5+ confirman dolor de no-shows y WhatsApp manual
- [ ] Citas textuales documentadas

### Beta Real (Fase C)
- [ ] 3-5 consultorios en beta gratuita
- [ ] 4 semanas de uso real
- [ ] Metricas no-shows antes/despues
- [ ] NPS de pacientes >= 30
- [ ] 2+ consultorios dispuestos a pagar post-beta

### Landing + Ads (Fase D)
- [ ] Landing page creada
- [ ] Campana FB/IG Ads ($1,500 MXN)
- [ ] Campana Google Ads ($1,000 MXN)
- [ ] 100+ visitantes
- [ ] >10% tasa de conversion

### Criterios para Pasar a P2
- [ ] 10+ entrevistas realizadas
- [ ] Beta completada con datos reales
- [ ] 2+ consultorios expresan disposicion a pagar
- [ ] Landing con >10% conversion
