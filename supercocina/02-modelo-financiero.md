## 1. Lean Canvas

| Bloque | Contenido |
|:-------|:----------|
| **Problema** | • Personas ocupadas (madres trabajadoras, empleados) sin tiempo para cocinar<br>• Opciones actuales son poco saludables, repetitivas o caras<br>• Falta flexibilidad horaria en servicios existentes<br>• Dolor recurrente 3-5 veces/semana |
| **Segmentos de Clientes** | **Primario:** "Laura" - Madres trabajadoras (35-42 años, 1-3 hijos, ingreso $15k-25k/mes)<br>**Secundario:** "Ricardo" - Empleados oficina/taller (28-45 años, ingreso $10k-20k/mes)<br>**Mercado:** 900 personas en Los Agaves, SLP (radio 3km) |
| **Propuesta de Valor Única** | Comida casera fresca y sana, preparada por cocineras locales, con sistema democrático de votación, precios dinámicos, y horarios flexibles (1pm-9pm). "El sabor de casa sin cocinar" |
| **Solución (MVP)** | • Dark kitchen + sistema pickup<br>• App/WhatsApp para pedidos mismo día (9am-1pm)<br>• 3 platillos + 3 guarniciones + 2 postres + 2 aguas diarias<br>• Cocineras a tiempo parcial<br>• Sistema de calificación y votación semanal<br>• Precio dinámico por horario |
| **Canales** | **Adquisición:** Volanteo en escuelas/oficinas, Facebook/Instagram Ads, WhatsApp<br>**Venta:** App móvil + WhatsApp Business<br>**Entrega:** Pickup principal (80%) + Uber Eats/Rappi opcional (20%)<br>**Retención:** Sistema de votación, programa de lealtad |
| **Flujos de Ingresos** | **Principal:** Venta transaccional de comidas<br>• Horario premium (1-5pm): $130/comida<br>• Horario regular (5-9pm): $100/comida<br>• Precio va bajando cada hora después de 5pm<br>**Futuro:** Suscripciones semanales con descuento |
| **Estructura de Costos** | **Variables (COGS):** Ingredientes, empaques, pago a cocineras (~60-65% del precio)<br>**Fijos (OpEx):** Renta cocina, salarios (1-2 personas), servicios, marketing<br>**Capital Inicial:** Equipamiento cocina, inventario inicial, permisos |
| **Métricas Clave** | • Comidas vendidas/día (meta: 20→40 en 6 meses)<br>• Tasa recurrencia (meta: 50%)<br>• NPS (Net Promoter Score)<br>• Margen de contribución por comida<br>• CAC (Costo Adquisición Cliente)<br>• Desperdicio de comida (meta: <5%) |
| **Ventaja Injusta** | • Red de cocineras locales (difícil de replicar)<br>• Datos de preferencias por votación (mejora continua)<br>• Sistema de pedido mismo día (elimina inventario)<br>• Comunidad early adopters en Los Agaves |

## 2. Modelo de Ingresos (Revenue Streams)

### Estructura de Precios Validada

| Tipo | Descripción | Precio | Frecuencia | Justificación |
|:-----|:------------|:-------|:-----------|:--------------|
| **Transaccional - Premium** | Comida completa horario pico | $130 | Por pedido | 28% de clientes validados dispuestos a pagar este precio. Comparable a comida de restaurante casual (~$150-180) |
| **Transaccional - Estándar** | Comida completa horario regular | $100 | Por pedido | 72% de clientes validados prefieren este precio. Comparable a comida rápida de calidad (~$80-120) |
| **Transaccional - Descuento** | Precio dinámico (después 7pm) | $80-90 | Por pedido | Maximiza ventas de inventario remanente. Benchmarks: apps como "Too Good To Go" (30-50% descuento) |
| **Delivery (opcional)** | Comisión plataforma | +20% | Por pedido con delivery | Uber Eats/Rappi comisión 18-25%. Costo absorbido por cliente |

### Proyección de Ticket Promedio

**Escenario Conservador (Mes 1-3):**
- 60% compran horario regular ($100) = $60
- 28% compran horario premium ($130) = $36.40
- 12% compran con descuento ($85) = $10.20
- **Ticket Promedio = $106.60**

**Escenario Estable (Mes 4-12):**
- 50% horario regular ($100) = $50
- 35% horario premium ($130) = $45.50
- 15% con descuento ($85) = $12.75
- **Ticket Promedio = $108.25**

### Métricas de Adquisición y Retención

| Métrica | Valor | Cálculo/Fuente |
|:--------|:------|:---------------|
| **CAC (Costo Adquisición Cliente)** | $110 | Marketing 10% × Ticket promedio × 10 compras para recuperar |
| **Frecuencia de Compra** | 3 veces/semana | Validado en entrevistas (3-5 veces/semana) |
| **Retención Estimada** | 50% después de 6 meses | Conservador para modelo de comida |
| **Vida del Cliente** | 24 semanas (6 meses) | Período de retención inicial |
| **Compras Totales por Cliente** | 72 compras | 3 compras/semana × 24 semanas |
| | | |
| **LTV (Lifetime Value)** | **$7,920** | 72 compras × $110 precio promedio |
| **CAC Total** | **$110** | 10% de primera compra como inversión adquisición |
| | | |
| **LTV:CAC Ratio** | **72:1** | $7,920 / $110 |
| **Payback Period** | **3 compras** | CAC / Margen contribución = $110 / $40.70 |
| **Margen Neto por Cliente (LTV)** | **$2,930** | (72 compras × $40.70) - $110 CAC |

### Análisis de Rentabilidad

**✅ Indicadores Saludables:**

1. **Margen de Contribución: 37%**
   - Benchmark industria food: 25-40%
   - SuperCocina está en rango superior

2. **LTV:CAC Ratio: 72:1**
   - Benchmark saludable: >3:1
   - SuperCocina: **EXCEPCIONAL** (72× retorno)
   - Razón: Alta frecuencia (3×/semana) + bajo CAC

3. **Payback Period: 3 compras (1 semana)**
   - Benchmark ideal: <12 meses
   - SuperCocina: **1 semana** = recuperación ultra rápida

4. **Margen Bruto: 47%**
   - Benchmark food delivery: 30-50%
   - SuperCocina: En rango alto gracias a modelo pickup

### Break-Even Operativo

**Supuestos de Costos Fijos Mensuales (a validar en Paso 4):**
- Estimado inicial: $30,000/mes

**Cálculo Break-Even:**
- Comidas necesarias/mes = Costos Fijos / Margen Contribución
- Break-Even = $30,000 / $40.70 = **737 comidas/mes**
- Break-Even diario (30 días) = **25 comidas/día**

**Validación con Escenario Pesimista:**
- Meta mes 1: 20 comidas/día = 600/mes
- **5 comidas/día por debajo** de break-even
- Déficit mes 1: ~$5,000 (requiere capital de trabajo)

**Validación con Escenario Realista:**
- Meta mes 2-6: 40 comidas/día = 1,200/mes
- **15 comidas/día por encima** de break-even
- Superávit mensual: ~$19,000
