# Lean Canvas: Tiendita Nocturna (v2)

| Bloque                  | Contenido                                                                                             |
| :---------------------- | :---------------------------------------------------------------------------------------------------- |
| **Problema**            | Falta de acceso a productos básicos de noche; inseguridad; soluciones actuales caras y lentas.          |
| **Segmento de Clientes**| "María", 32-42 años, profesionista con hijos en zonas residenciales.                                  |
| **Propuesta de Valor Única** | "Tus compras de conveniencia y medicinas básicas en tu puerta en menos de 30 min, 24/7."             |
| **Solución (MVP)**      | App móvil con: 1. Catálogo de inventario en tiempo real. 2. Carrito de compras. 3. Pasarela de pagos en línea. |
| **Canales**             | Anuncios en redes sociales (grupos de colonias), alianzas con administradores de condominios.          |
| **Flujos de Ingresos**  | 1. Margen sobre producto vendido. 2. Tarifa de envío ($25).                                           |
| **Estructura de Costos**| Costo de productos (COGS), repartidores, software de la app, marketing, local para la dark store.       |
| **Métricas Clave**      | 1. Tasa de conversión (15.65%). 2. Monto de venta semanal. 3. Número de pedidos por semana.             |
| **Ventaja Injusta**     | Algoritmos y automatización de logística interna para garantizar precios bajos y entregas rápidas.     |
