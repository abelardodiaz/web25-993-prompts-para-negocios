## Estado: P0-pendiente

Evaluacion de nichos para tiendas online con catalogo Syscom en Shopify.

## Contexto

Convergencia de 2 assets existentes:
- **Proyecto 019** (tienda.redv6.com): Integracion Syscom API funcional

## Hallazgo Clave

El sync Syscom API esta commoditizado (MojoMexico, Kaesys KWS1, DecodeCMS ya lo ofrecen). El diferenciador real es **expertise vertical + servicio completo**, no el tech.
