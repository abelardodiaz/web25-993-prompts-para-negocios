## Competencia Descubierta (Investigacion Feb 2026)

### HALLAZGO CRITICO: El sync API esta commoditizado

La investigacion revelo que multiples empresas ya ofrecen sincronizacion Syscom API como servicio. El tech no es diferenciador.

### Competidores Directos (Sync API)

| Competidor | Producto | Plataformas | Precio | URL |
|-----------|----------|-------------|--------|-----|
| **MojoMexico** | Syscom API sync (50,000+ productos, fotos, specs, stock, precios especiales) | Shopify, WooCommerce, Magento 2, PrestaShop | Renta mensual | aum.mojomexico.net, syscom.mojomexico.net |
| **Kaesys KWS1** | Puente multi-mayorista (Syscom, CVA, Tecnosinergia, TVC, CT, PCH, Ingram, Mouser) | Shopify, WooCommerce, PrestaShop, MercadoLibre, Amazon | Por cotizacion | kaesys.com/kws1, nubekaesys.com/kws1 |
| **DecodeCMS** | SYSCOMWOO - Plugin WordPress/WooCommerce | WooCommerce | Plugin | decodecms.com/syscom |

**MojoMexico** es el competidor mas directo: ya ofrece exactamente "Syscom API -> Shopify" con sync completo de catalogo. Tambien cubre CVA, CT, TVC, PCH, Intcomex.

**Kaesys KWS1** va mas alla: conecta multiples mayoristas a multiples plataformas, incluyendo marketplaces (MercadoLibre, Amazon).

### Implicacion Estrategica

**El tech (API sync) NO es el diferenciador** - esta commoditizado. Los diferenciadores reales son:

1. **Expertise vertical**: Contenido tecnico, guias de compatibilidad, soporte pre-venta
2. **Catalogo curado**: No 75,000 productos sino los 500 correctos para el nicho
3. **Servicio completo**: Tienda + marketing + operaciones, no solo un plugin
4. **Shopify-native**: La mayoria de competidores son WooCommerce-first

## 3 Modelos de Negocio (Revisados Post-Competencia)

### Modelo A: Servicio "Tienda Llave en Mano"

**Que es:** Ofrecer a distribuidores Syscom una tienda Shopify completa, no solo el sync API.

**Incluye:**
- Tienda Shopify con diseno profesional
- Catalogo curado para su vertical (no dump de 75K productos)
- Contenido tecnico (fichas, guias de compatibilidad, comparativas)
- Capacitacion del equipo
- Soporte tecnico + marketing inicial

**Diferenciacion vs MojoMexico/Kaesys:** Ellos venden el plugin/sync. Nosotros vendemos el resultado: una tienda que vende. Expertise vertical que una empresa de software no tiene.

**Riesgo:** Escala limitada (servicio, no producto). Requiere ventas consultivas.

### Modelo B: Tienda Propia Especializada

**Que es:** Operar una tienda online propia con catalogo Syscom curado en un vertical especifico.

**Diferenciacion vs competidores:**
- Catalogo curado (300-500 productos, no 75,000)
- Contenido tecnico experto (guias de instalacion, calculadoras, compatibilidad)
- Soporte pre-venta con conocimiento real
- Experiencia Shopify superior (la mayoria de competidores usan WooCommerce basico)

**Vertical inicial:** CCTV/Seguridad o Solar Residencial (no ambos)

**Riesgo:** Competir con Syscom.mx directo y distribuidores existentes. Requiere marketing + operaciones.

### Modelo C: Hibrido (Recomendado para validacion)

**Que es:** Montar tienda propia como showcase, luego ofrecer el servicio llave-en-mano a otros distribuidores.

**Ventaja:** La tienda propia demuestra capacidad real. "Mira lo que hicimos para nosotros, te lo hacemos a ti."

**Riesgo:** Dividir foco entre operar tienda + vender servicio.

## El Problema que Resuelve

### Para el Distribuidor Syscom (Modelo A/C)

Los distribuidores Syscom quieren vender online pero:

1. **MojoMexico/Kaesys les dan el sync, pero no la tienda**: Quedan con 50,000 productos sin curar, sin contenido, sin marketing
2. **No tienen capacidad de e-commerce**: Son empresas de instalacion/distribucion, no de marketing digital
3. **WooCommerce es complejo**: La mayoria de soluciones son WC-first, que requiere mas mantenimiento tecnico
4. **Sin contenido tecnico**: Fichas genericas de Syscom, sin valor agregado

### Para el Comprador Final (Modelo B/C)

Profesionales de CCTV/solar que compran equipo:

1. **Syscom.mx es mayorista**: Requiere cuenta, no atiende compras pequenas o individuales
2. **Tiendas existentes son basicas**: Sin contenido tecnico, sin guias, sin soporte pre-venta
3. **No hay curaduria**: 75,000 productos sin orientacion de que comprar para cada caso
