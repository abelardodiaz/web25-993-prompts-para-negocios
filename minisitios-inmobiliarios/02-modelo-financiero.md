# Modelo Financiero: Minisitios Inmobiliarios

## 1. Lean Canvas

| Bloque | Contenido |
| :--- | :--- |
| **Problema** | Pequeños administradores pierden prospectos por ineficiencia de WhatsApp y falta de imagen profesional. Gestión de fotos tediosa. |
| **Cliente** | Administradores independientes y dueños con 1-10 propiedades en México (Inicio: SLP). |
| **Propuesta de Valor** | Minisitio profesional entregado en 8 días. Servicio "Concierge": tú mandas fotos por WhatsApp, nosotros hacemos todo. |
| **Solución** | Plantilla web optimizada, dominio propio, correo profesional, integración directa a WhatsApp. |
| **Canales** | Venta directa (WhatsApp), Grupos de Facebook, Marketplace, Referidos. |
| **Ingresos** | Pago único por Setup ($8,000) + Anualidad de mantenimiento ($2,000). |
| **Costos** | Dominio, Hosting, Licencias de correo, Tiempo de implementación (Mano de obra). |
| **Métricas Clave** | # Sitios vendidos/mes, Tasa de renovación anual, Tiempo de entrega. |
| **Ventaja Injusta** | Fundador en el gremio (entiende el dolor), Servicio 100% gestionado (cero curva de aprendizaje para el cliente). |

## 2. Modelo de Ingresos (Revenue Streams)

| Tipo | Concepto | Precio (MXN) | Frecuencia | Justificación |
| :--- | :--- | :--- | :--- | :--- |
| **Setup Fee** | Creación Minisitio + 10 Props | **$6,000** | Único | Precio competitivo para entrada rápida al mercado. |
| **Recurrente** | Hospedaje, Dominio, Soporte | **$1,000 (Año 1) / $2,000 (Año 2+)** | Anual | Barrera de entrada baja el primer año. |
| **Promo Validación** | "Descuento Fundador" | **$4,000** | Único | Solo para los primeros 5 clientes. |

## 3. Unit Economics (Los Números que Importan)
- **CAC (Costo Adquisición Cliente):** $0 (Orgánico inicial)
- **LTV (Valor del Cliente a 3 años):** $6,000 (Setup) + $1,000 (Año 1) + $4,000 (Años 2-3) = **$11,000**
- **Margen Bruto (Precio Normal):** 66% ($4,000 por venta de $6,000)
- **Margen Bruto (Promo):** 50% ($2,000 por venta de $4,000)
- **Punto de Equilibrio Mensual:** 1.1 ventas (Precio Promo) / 0.73 ventas (Precio Normal)

## 4. Costos de Arranque (CapEx)
- **Inversión Inicial:** **$0 MXN**
- **Estrategia:** Uso de activos existentes (VPS, Plantilla, Laptop).

## 5. Estructura de Costos Fijos (OpEx)
- **Burn Rate Mensual:** **$4,400 MXN**
- **Desglose:** Publicidad ($2,000) + Herramientas AI ($2,400).

## 6. Proyección Financiera a 12 Meses
- **Ventas Totales:** 24 sitios
- **Ingresos Totales:** $158,000 MXN
- **Utilidad Neta:** **$89,600 MXN** (57% Margen Neto)
- **Flujo de Caja:** Positivo desde el Mes 1.

## Veredicto Final del Arquitecto
- **Capital Total Requerido:** $0 (Se financia con la primera venta).
- **Salud del Modelo:** **EXCELENTE**.
  - Optimización de costos de IA y publicidad mejora la utilidad en un 41%.
  - Flujo de caja muy saludable.
  - Riesgo mínimo.
- **Recomendación:** Proceder inmediatamente a la ejecución de ventas.
